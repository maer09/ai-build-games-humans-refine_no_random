# 模板范围（仅不变层）

此捆绑的 `godot-base/` 模板是一个纯基础设施骨架。
不得锁定游戏玩法、视觉身份或音频身份。

## 允许

- `main.gd` 中的测试钩子接口
- 通用遥测/KPI 辅助工具
- 底层音频原语（`sine/square/triangle/noise`，包络）
- Headless/测试/导出基础设施

## 禁止

- 默认游戏逻辑（得分获取、失败条件、敌人行为）
- 默认视觉风格（调色板、发光配置、HUD 语言、脚本 FX）
- 与游戏语义绑定的、事件级别的音效词汇
- 预捆绑的字体资源和字体许可证文件
- 基础接口中领域锁定的命名（如固定的 `left/right/thrust`、`enemy/chaser` 语义）
- 假设特定控制模式的预定义探索性输入编排

## 实现说明

从此模板创建新游戏时，先在游戏特定的文件中实现机制/视觉/音频，
保持模板本身不包含游戏特定的行为。
